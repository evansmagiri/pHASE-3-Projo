# Phase 3 Project: Financial Inclusion in Kenya

Predicting bank account ownership to help banks/fintech target unbanked Kenyans.

## Key Findings
- Rural and no-cellphone access = biggest barriers
- Model catches 76% of unbanked for outreach
- Best model: Logistic Regression
